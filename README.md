# Everpoint
 API for junior test

### Installation

Everpoint requires [Docker](https://www.docker.com/) to run.

```sh
$ cd Everpoint
$ docker compose-up
```

##### The folder for .xlsx is [media/files/](https://github.com/toren332/Everpoint/tree/master/media/files)

### Tests

```sh
$ cd Everpoint
$ pytest
```
